# 项目描述

–实现了微信小程序的天气预报、每日笑话、添加股票、星座运势、图片备份等功能

–后端使用 Django 框架，前端使用 WSXML、WXML和JS ，前后端分离，减少重复开发，代码易于维护

–数据存储使用 MySQL，ORM 使用 Django ORM

–使用 RESTful API 设计 Django 路由

–使用 Shell 脚本进行一键部署
